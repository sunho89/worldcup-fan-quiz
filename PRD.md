# 🏆 世界杯球迷人格测试 — 产品需求文档

---

## 1. 项目亮点

### 解决什么问题

| 痛点 | 解决方案 |
|------|----------|
| 世界杯期间社交货币匮乏 | 一张专属"球迷身份证"，快速建立看球社交话题 |
| 伪球迷尴尬 | 用测试结果自嘲式认领"懂王型/云球迷"，化解身份焦虑 |
| 缺乏记忆点的互动 | 弹幕式动效 + 定制化卡片，让分享自带传播属性 |

### 好玩在哪里
- **梗文化密集**：题目大量使用球迷黑话（"我奶奶都能进的球"、"越位型人格"）
- **社交货币感强**：每种球迷类型都有精准吐槽金句，支持一键复制文案
- **彩蛋设计**：答完题后不立即显示结果，先来一段"命运揭示"动画再出答案
- **世界杯限定感**：背景动态效果模拟赛场氛围（绿色草坪纹理、黑白五边形足球元素滚动）

## 2. 核心功能与用户流程

```
[启动页/封面] → [引导页-规则说明] → [答题页 x10] → [命运揭示动画 3s] → [结果页] → [分享卡片生成] → [二次分享/重新测试]
```

### Step 1: 启动页/封面
- 全屏 H5 页面，中央大字"你是哪种球迷？"
- 背景：世界杯官方配色 + 动态足球元素漂浮
- 底部 CTA 按钮："开始测试"（点击有触感反馈动效）
- 顶部小字："已有 1,234,567 人揭晓身份"

### Step 2: 引导页（规则说明）
- 3 条简短规则，配合球迷风格插画：
  - "共 10 道题，答完自动出结果"
  - "没有标准答案，只有你的真实人格"
  - "支持分享，但禁止剧透给朋友"
- 按钮："我准备好了，开整！"

### Step 3: 答题页（循环 10 次）
- 单题单页，避免用户疲惫
- 顶部：进度条 + 当前题号 "3/10"
- 题目：3-4 个选项，文字简短有梗
- 每个选项点击后有"选中高亮"动效，然后自动滑入下一题（800ms 延迟）
- 支持左滑/右滑选择（移动端手势）

### Step 4: 命运揭示动画（3 秒）
- 全屏渐黑背景
- 中央出现旋转的足球 + 光效
- 文字："正在分析你的球迷DNA..."
- 3 秒后自动跳转结果页

### Step 5: 结果页
- 超大标题：球迷类型名称 + emoji（如 "懂王型球迷 🧠"）
- 匹配球队：显示 1-2 支"本命球队"（如：德国 / 西班牙）
- 类型描述：3-4 句话精准吐槽
- 雷达图/能力值展示（如：足球知识 85 / 看球热情 60 / 赌性 40）
- 底部按钮："生成专属卡片" / "分享给朋友" / "重新测试"

### Step 6: 分享卡片
- 自动生成一张横版卡片（适配朋友圈/微博）
- 包含：球迷类型 + 匹配球队 + 一句金句 + 小程序码（可选）
- 支持保存到相册或直接分享

## 3. 问题设计（10 道）

### 设计原则
- 每道题聚焦一个维度，避免一道题测多个特质
- 选项要有"梗"，让用户觉得有趣而非审问
- 正确答案不明显（没有社媒上公认的"正确球迷形象"）

## 4. 球迷类型定义（8 种）

### 🏆 豪门铁粉
**匹配球队**：巴西 / 德国 / 法国 / 西班牙

**描述**：
支持豪门多年，见证过王朝更迭。你不是墙头草，是真正的底蕴球迷。半夜三点爬起来看球是家常便饭，输球时你比球员还难受，赢球时你比球员还先庆祝。

**金句**：*"我们不是赢在现在，是赢在底蕴。"*

**能力值**：足球知识 90 | 看球热情 95 | 社交分享 60 | 赌性 30

---

### 🧠 懂王型
**匹配球队**：意大利 / 阿根廷 / 英格兰

**描述**：
你嘴上懂球，肚子里也确实懂。越位规则能讲十分钟，补时阶段能预测换人。朋友看球遇到问题第一个问你，你是行走的足球百科，也是朋友圈的战术大师。

**金句**：*"这球索斯盖特必须背锅。"*

**能力值**：足球知识 95 | 看球热情 75 | 社交分享 80 | 赌性 45

---

### 📊 数据流
**匹配球队**：英格兰 / 比利时

**描述**：
你看的不是球，是数据。xG、传球成功率、压迫次数张口就来。你可能不是最狂热的球迷，但你一定是最"理性"的球迷。比赛结果可以输，但数据预测不能错。

**金句**：*"这场预期进球是 2.3，你跟我说 1-0？"*

**能力值**：足球知识 85 | 看球热情 60 | 社交分享 70 | 赌性 80

---

### 🎉 氛围组
**匹配球队**：随时变，谁赢支持谁

**描述**：
你是世界杯期间最可爱的人。酒吧有你就热闹，集锦有你就完整，朋友圈有你就欢乐。你可能记不清越位规则，但你能记住每场比赛的最佳庆祝动作。

**金句**：*"今晚西班牙对日本？走！反正我支持好看的！"*

**能力值**：足球知识 40 | 看球热情 85 | 社交分享 95 | 赌性 50

---

### 🎰 赌神型
**匹配球队**：随缘（赔率决定主队）

**描述**：
世界杯期间，你的生活围绕菠菜转。每场比赛都有你的投注，单关、串子、比分、角球数，你样样精通。赢了会所嫩模，输了下海干活。足球在你眼里不只是运动，还是一场大型经济行为。

**金句**：*"这场不博一下对不起看球的时间。"*

**能力值**：足球知识 75 | 看球热情 90 | 社交分享 50 | 赌性 100

---

### ☁️ 云球迷
**匹配球队**：只认识姆巴佩和梅西

**描述**：
你很忙，忙到没时间看完整场比赛。但你很会冲浪，微博热搜、抖音集锦、朋友圈截图，让你即使不打开电视也能掌握所有名场面。你是互联网足球的原住民，也是足球梗文化的最大传播者。

**金句**：*"我就看了集锦，这球姆巴佩太牛了。"*

**能力值**：足球知识 35 | 看球热情 55 | 社交分享 90 | 赌性 25

---

### 🧘 佛系养生球迷
**匹配球队**：北欧球队 / 中小国家队

**描述**：
胜负在你眼里皆是浮云，看球就是为了开心。你不为输赢动气，不和球迷吵架，主队输了你就去散步。别人问你为什么支持冰岛，你说我就是喜欢他们的团队精神。

**金句**：*"足球嘛，图个乐子，何必呢。"*

**能力值**：足球知识 60 | 看球热情 50 | 社交分享 45 | 赌性 15

---

### 📺 战术分析师
**匹配球队**：意大利 / 德国

**描述**：
你看的是球，想的是战略。瓜迪奥拉的战术板你都能画，克洛普的高位压迫你能写篇论文。你不是普通球迷，你是潜伏在看球人群中的教练。每进一球，你首先想到的是这球是怎么跑位的。

**金句**：*"这球不是个人能力问题，是体系问题。"*

**能力值**：足球知识 98 | 看球热情 80 | 社交分享 65 | 赌性 35

## 6. 技术方案

### 方案选型

#### 纯前端方案（推荐 v1）
- **前端框架**：React + TypeScript 或 Vue 3 + TypeScript
- **构建工具**：Vite
- **样式方案**：Tailwind CSS + CSS Variables
- **动效库**：Framer Motion（React）或 GSAP（通用）
- **状态管理**：Zustand 或 Pinia
- **发布平台**：微信小程序 H5 / 抖音 H5 / 直接 URL 访问

**优势**：成本低、迭代快、无后端依赖
**局限**：无法做排行/用户持久化

#### 全栈方案（推荐 v2）
- **前端**：同上
- **后端**：Node.js + Express 或 Spring Boot
- **数据库**：Redis（存储答题结果/排行）
- **存储**：COS/C OSS（分享卡片图片持久化）

### 核心模块

| 模块 | 技术方案 | 说明 |
|------|----------|------|
| 答题逻辑 | 前端状态机 | 每题独立 state，答题后累加维度分 |
| 结果计算 | 加权算法 | 10题 x 多维度 = 8种球迷类型 |
| 卡片生成 | html2canvas / Canvas API | 前端直接生成图片 |
| 动画效果 | GSAP + CSS Animation | 高性能动画 |
| 音效（可选） | Howler.js | 进球/揭示时的音效反馈 |

### 答题算法伪代码

