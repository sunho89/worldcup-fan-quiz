# ⚽ 世界杯球迷人格测试

> 通过 10 道有趣的选择题，测出你是哪种球迷，匹配你的"本命球队"。

**2026 世界杯特别企划** | 知乎「足球季 Vibe Coding 挑战赛」参赛项目

---

## 🎯 项目亮点

| 亮点 | 说明 |
|------|------|
| 社交货币 | 一张专属"球迷身份证"，发朋友圈/群聊自带话题 |
| 梗文化密集 | "越位型人格"、"我奶奶都能进的球"等球迷黑话 |
| 彩蛋动画 | 答题后先看"命运揭示"动画，仪式感拉满 |
| 分享卡片 | 自动生成可保存/分享的结果卡片 |

---

## 🕹️ 玩法说明

1. **打开即玩**：无需注册，打开链接直接开始
2. **回答 10 道题**：每道题对应一个球迷维度
3. **等待命运揭示**：3 秒动画分析你的"球迷DNA"
4. **获得结果**：你是 8 种球迷类型中的哪一种
5. **生成卡片**：一键保存/分享专属结果图
